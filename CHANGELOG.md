## [2.0.0] - 2026-04-11

### Added
- Initial release of ANDL 2.0 protocol specification
- Vector-based message format (128-bit semantic vector)
- Support for 8 message types (INTENT, DATA, QUERY, RESPONSE, CONTROL, HEARTBEAT, CONTEXT, ERROR)
- Semantic encoding system (Task, Target, Attribute, Domain)
- Reference implementation in Python
- First AI-to-AI vector communication experiment
- Vector Translator specification for human supervision
- Security and governance framework
- Comprehensive documentation and examples

### Experimental
- World's first AI-to-AI vector communication
- 20 rounds of bidirectional communication
- 68.5% bandwidth savings verified
- Zero human language intervention
